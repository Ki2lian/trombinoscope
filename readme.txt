Ce dossier contient 2 autres dossiers.
Un dossier etudiant pour le site pour les étudiants et l'autre pour l'équipe pédagogique.
Vous devez mettre le contenu des fichiers dans 2 sites (hébergeurs) différents.
Il y a un fichier config.php sur les deux sites, veuillez consulter ce dernier pour modifier si besoin.
Le site n'est pas responsive, je ne me suis pas encore attaqué plus en profondeur sur bootstrap pour pouvoir l'utilisé.

Démo:
Site étudiant -> https://etudiants.alwaysdata.net/
Site équipe pédagogique -> https://equipe-pedagogique.alwaysdata.net/
